Mock Image 有图有真相，快速生成你所需要的图片。

简介

还在为开发过程中找固定尺寸的图片而发愁或者默默地准备自己切一张?
该使用MockImage了，支持多种图片格式，多种生成参数。
了解详情： 使用文档、玩一下。

修改自 dummyimage.com/ by soulteary。

程序功能

  • 生成指定尺寸的图片
  • 指定生成图片格式
  • 指定生成图片颜色
  • 指定图片文本

使用文档

尺寸

宽度 x 高度

  • 高度是可选属性，如果没有设置宽度，那么将生成一个正方形。例如： http://dummyimage.pantimos.io/300
  • 尺寸参数必须作为URI中的第一个参数。
  • 你可以只指定其中一个参数，另外一个使用比例来代替，程序将会自动计算该比例的具体数值。例如： http://dummyimage.pantimos.io/640x4:3 或者 http://dummyimage.pantimos.io/16:9x1080

颜色

背景色 / 文本颜色

  • 颜色数值推荐使用HEX格式，例如：fff，ffffff。
  • 颜色必须跟在尺寸之后，例如： http://dummyimage.pantimos.io/250/ffffff/000000
  • 背景色是第一个参数，文本颜色是第二个参数。
  • 背景色和文本颜色皆为可选参数，默认背景色为#ccc，文本颜色为000。
  • 下面是一些颜色的缩写例子：
    • 3位颜色数值将会扩展到6位： 09f => 0099ff
    • 2位颜色数值将会扩展到6位： ef => efefef
    • 1位颜色数值将会扩展到6位： c => cccccc

图片格式

.gif, .jpg, .png

  • 添加不同的文件格式名称，将会输出不同格式的图片。
  • 默认图片格式为gif。
  • jpg和jpeg输出内容相同。
  • 格式名称必须在地址的最后面，
    • http://dummyimage.pantimos.io/300/09f/fff.png

文本内容

?text=Hello+World

  • 自定义文本内容，必须在请求URI的末端。
  • 默认内容为图片尺寸，Width x Height。
  • 支持中英文。
  • 空格 => +
    • http://dummyimage.pantimos.io/200x300?text=HELLO+world+rocks!
